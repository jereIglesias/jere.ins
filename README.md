# App Avengers: Endgame

Aplicación móvil simple (React Native) que muestra información de la película "Avengers: Endgame" usando la API de TMDB.

Instalación rápida:

1. Coloca tu API key en `src/config.ts` (reemplaza el valor de `API_KEY`).
2. Instala dependencias:

```
npm install
```

Ejecución (Android):

```
npx react-native run-android
```

Notas:

- Asegúrate de tener instalado JDK y Android Studio con un emulador o un dispositivo conectado.
- Este README es intencionalmente simple.
